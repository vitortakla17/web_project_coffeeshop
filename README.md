# Projeto 4: Triple Espresso

A página da cafeteria da Triple Espresso é o quarto projeto no programa de desenvolvimento web da TripleTen. Nela fora aplicado recursos mais avancados de HTML e CSS, como: definicao dos estilos do navegador, adicionar fontes, transbordamento de bloco e utilizacao das pseudoclasses, pseudoelementos e seletores de atributos.
Tambem foi aplicado incorporacao de conteudo e formularios. Alem disso, os arquivos do documento foram organizados conforme a metodologia BEM flat.

## Recursos do projeto

- HTML5 semântico
- Flexbox
- Posicionamento
